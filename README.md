# amazon-clone
A clone of the amazon website with django
## Installation
```bash
$ git clone https://github.com/emaribeyodn/amazon-clone
$ cd amazon-clone
$ python3 -m venv .env
$ source .env/bin/activate
$ pip install -r requirements.txt
$ python3 manage.py migrate
$ python3 manage.py runserver
```
